# Rotbak-Cashier
Cara buka project:
1. Buka Netbeans Terlebih dahulu
2. Open Project rotbak31_premium
3. Tambahkan library (Masukkan semua file .jar dalam folder Library) ke dalam project
3. Masukkan Database ke dalam PHPMyAdmin
4. Jalankan Program
